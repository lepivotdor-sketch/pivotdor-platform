Sécurité de l’accès

Cette version ne doit pas contenir de vrais liens privés vers les formations payantes. Les vrais accès doivent être protégés côté serveur. Aucune donnée bancaire, aucun vrai mot de passe et aucun fichier payant complet ne doivent être placés dans cette page publique.

GitHub Pages est excellent pour l’interface, mais pas pour cacher du contenu payant complet.
